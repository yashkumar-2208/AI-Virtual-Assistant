# Mahiru - AI Virtual Assistant

Mahiru is a Python-based AI Virtual Assistant that listens to your voice, understands commands, and responds using speech.

# Features

- Speech Recognition
- Text-to-Speech
- Open websites
- Voice Conversation
- JSONL Dataset Support
- Open And Create File



# Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- pandas
- nltk
- prompt_toolkit

cd Mahiru

python -m nltk.downloader punkt stopwords
